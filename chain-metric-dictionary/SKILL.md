# 连锁指标口径查询

## 使用方式

当收到指标口径查询请求时，按以下步骤处理：

1. 确定用户需要查询的指标类型（原子指标、衍生指标、比例指标还是效率指标）
2. 从 references/ 中选择对应的参考文件加载
3. 根据字典内容给出精确的指标定义和计算公式
4. 如需要，提供 SQL 示例并注明数据源

## 核心使用流程

### 1. 查询指标定义时
- 从 references/连锁经营指标口径回答.md 中查找指标ID和定义
- 如需SQL实现，打开 指标字典.xlsx 的对应工作表
- 如需了解数据表结构，打开 数据库表！.docx

### 2. 回答口径问题时
- 优先使用原子指标定义（A001-A019）作为基准
- 衍生指标 = 原子指标 + 维度切片
- 注意区分三层口径：整体 / 非0元 / 非0非大师
- 引用参考值时要注明数据来源

### 3. 编写SQL时
- 参考指标字典中的SQL示例
- 确认正确的数据源表和数据分区（dp = T-1）
- 确认正确的过滤条件（is_valid=1 等）
