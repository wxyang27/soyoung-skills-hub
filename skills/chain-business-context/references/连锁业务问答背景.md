## 五、指标体系框架

### 核心公式

`
收入 = 核销人次 x 客单价                              （人视角 - 量价拆解）
收入 = 核销服务点数 x 单服务点收入                      （货视角 - 货盘拆解）
连锁总收入 = 医美收入 + 好物收入                       （总盘拆解）
医美收入 = 新客收入 + 老客收入                         （人群拆解）
新客收入 = 私域 + 老带新 + 公域(美团+小红书+KOL+线下)   （渠道拆解）
老客收入 = 活跃老客(85%+) + 非活跃老客(~10%) + 当月核销新客(~5%)

时间进度达成率 = 实际达成 / (目标 x 时间进度%)
毛利率 = (收入 - 供应链成本) / 收入      目标 48%
获客成本占收比 = 获客总成本 / 新客收入   参考 30-34%
公域ROI = 新客核销收入 / 获客成本        目标 >= 1.4
活跃到院率 = 当月到院活跃用户 / 活跃用户池  目标 16-17%
升单率 = 到院后发生升单的用户 / 总到院用户
品项渗透率 = 品项核销人数 / 总核销人数

### 单服务点收入三层口径
- **整体**: 含0元、含大师团
- **非0元**: 剔除0元单
- **非0非大师**: 剔除0元单和大师团 —— 最真实反映核心品项经营质量

### 参考值速查

| 指标 | 参考值 |
|------|--------|
| 活跃用户到院率 | 16-17% |
| 公域ROI | >= 1.4 |
| 获客成本占收比 | 30-34% |
| 0元单占比 | 实际18-24% / 行业5-15% |
| 定价/活动价/大促价 | 理想5:3:2 / 实际2:3:5 |
| 品项毛利率目标 | 48%（大单品>=70%、利润品>=60%、常规品>=50%）|
| 新客收入占比 | ~35% |
| 老客收入占比 | ~65% |
| 大单品收入占比 | 34-50%（已破50%）|
| 大师团收入占比 | 2-10%（持续下降）|
| CR30收入占比 | ~83% |
| 老带新获客成本 | ~300-400/人 |
| 美团获客成本 | ~500-900/人 |
| 老带新付费转化率 | ~48% |
| 老带新裂变系数 | ~2.1 |

## 六、经营分析核心方法论

### 回答的四个问题
1. 现在业务离目标还差多少？
2. 差距出在规模、价格、结构还是履约？
3. 问题集中在哪些渠道、城市、门店、品项或用户层？
4. 接下来最有效的追数动作是什么？

### 五层分析框架（总盘 + 人货场财）

| 层级 | 核心关注 |
|------|---------|
| 总盘 | 总收入、总利润、总达成率、时间进度达成率 |
| 人 | 新客(分渠道)、老客(活跃/流失)、老带新、活跃池、复核 |
| 货 | 品项结构(大单品/常规品/大师团)、单服务点收入、毛利率 |
| 场 | 门店、城市、大区、供给/体验/效率、升单 |
| 财 | 毛利、费用、占收比、利润、预算差异 |

### 七步标准分析顺序
1. 先看时间进度达成
2. 再看总收入和新老客结构
3. 再做量价拆解（核销人次 x 客单价）
4. 再看支付到核销的过程转化
5. 再看品项结构（大单品/常规品/大师团）、价格结构、0元单结构
6. 再下钻城市、门店、渠道、品项
7. 最后输出动作方案

### 五套核心分析逻辑

**1. 时间进度分析法**
- 当前时间进度是多少？
- 收入进度是多少？
- 已预约预估能补到多少？
- 全月预估离目标还差多少？
- 剩余GAP靠什么补？

**2. 量价拆解法**
- 规模掉了还是价格掉了？
- 是短期活动影响还是长期结构变化？
- 是整体都弱还是某几个维度在拖累？

**3. 支付到核销转化法**
- 支付好、核销慢 -> 兑现滞后，后续仍有释放空间
- 支付一般、核销好 -> 存量池在释放，关注新增支付
- 支付和核销都弱 -> 全链路诊断

**4. 结构分析法**
- 0元单结构（占比趋势、集中品项、是否有高耗）
- 价格带结构（标准定价/活动价/大促价比例）
- 品项结构（大单品/常规品/大师团占比变化）
- 渠道结构（私域/老带新/公域各自贡献占比）
- 新老客结构（新客~35% / 老客~65%）

**5. 城市门店下钻法**
- 先看大区（华东/华南/华中/华北）
- 再看城市（仅展示红灯城市）
- 再看门店（Bottom3门店）
- 再看门店内部的渠道、品项、体验和效率

## 七、异常诊断路径

### 收入不达预期
时间进度达成率 -> 新客还是老客拖累 -> 人次还是客单拖累 -> 是否有活动/大师团/节假日 -> 是否集中在某区域或门店

### 新客收入弱
私域支付是否不足 -> 老带新质量是否下降 -> 公域投放是否缩量或转化下降 -> 价格带是否向低价倾斜 -> 主推品客单过高

### 老客收入弱
活跃用户池增速是否放缓 -> 到院率是否下降 -> 待核销订单是否不足 -> 升单率和升单客单价是否下降

### 单服务点收入下降
0元单占比是否上升 -> 低价品是否放量 -> 活动价/大促价占比是否过高 -> 大师团结构是否变化

### 毛利率下降
高耗材占比是否上升 -> 赠单中是否出现高耗品 -> 低毛利品是否放量

### 核销人次突然下降
整体人次/人均服务点/客单三方向 -> 拆新老客 -> 新客按渠道拆 -> 老客按城市拆 -> 按支付间隔拆（当日/近1周/近2-3周/近10周/10周以上）

## 十、报表建议结构

### 日报结构
1. 整体大盘：医美收入(日/月环比、时间进度达成率) + 量价(客单价、核销人次) + 渠道占比
2. 用户拆解：新客(收入、客单价、渠道占比) + 老客(收入、活跃池占比、到院率)
3. 品类拆解：大单品(占比+Top3) + 大师团(占比+Top3) + 常规品(占比+Top3)
4. 大区拆解：华东/华南/华中/华北各自收入+环比+进度+Bottom3

### 每页回答三个问题
- **发生了什么**（结果判断）
- **为什么发生**（原因拆解）
- **接下来怎么办**（动作建议）

### 每个异常回答四个问题
1. 差额是多少？
2. 差在规模、价格还是结构？
3. 集中在哪个渠道/品项/区域/门店？
4. 下周靠什么追回来？
