# ② 维度与值域词典

> 将"私域""新客""大单品"等维度语言映射到 `字段 = 值`

## 渠道维度

### 一级渠道 (cx_first_channel)

| 业务名称 | 别名 | 字段值 |
|---------|------|--------|
| 私域 | 私有渠道、private | 私域 |
| 公域 | 公开渠道、public | 公域 |
| 老带新 | 转介绍、referral | 老带新 |

### 二级渠道 (cx_second_channel) — 优先使用

| 业务名称 | 字段值 | 近3月收入占比 |
|---------|--------|-------------|
| 私域活动 | 私域活动 | 26.3% |
| 老带新 | 老带新 | 25.1% |
| 公域 | 公域 | 23.5% |
| 小程序单品牌 | 小程序单品牌 | 7.2% |
| 自媒体口碑 | 自媒体口碑 | 4.7% |
| 微信KOL | 微信KOL | 2.5% |
| 微信信息流 | 微信信息流 | 2.3% |

**渠道选择原则：** 默认用 cx_second_channel（28个值），需要概览时用 cx_first_channel（3个值），需要最细粒度用 cx_third_channel。

## 新老客维度 (is_new)

| 业务名称 | 字段值 | 说明 |
|---------|--------|------|
| 新客 | is_new=1 | 全历史首次核销 |
| 老客 | is_new=0 | 非首次核销 |
| 全部 | is_new IN (0,1) 或不加过滤 | 所有客户 |

| 品项 | 收入占比 |
|------|---------|
| 韩国进口婕尔玻尿酸-颈部-1ml | TOP1 |

| 常规品项-50U | TOP3 |

**查询建议：** 品项名含特殊字符，SQL 用 LIKE '%关键词%'，不用精确匹配。

## 时间维度映射

| 用户说法 | SQL 表达 |
|---------|---------|
| 昨天 | executed_date = DATE_SUB(CURRENT_DATE, 1) |
| 今天 | executed_date = CURRENT_DATE |
| 本周 | executed_date >= 本周一 AND <= 今天 |
| 上周 | executed_date >= 上周一 AND <= 上周日 |
| 本月 | executed_date >= 本月1日 AND <= 今天 |
| 上月 | executed_date >= 上月1日 AND <= 上月最后一天 |
| 近3个月 | executed_date >= DATE_SUB(CURRENT_DATE, 90) |
| 近7天 | executed_date >= DATE_SUB(CURRENT_DATE, 7) |
| 同比 | executed_date BETWEEN 去年同期的起止日期 |
