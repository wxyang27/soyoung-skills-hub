## 二、衍生指标（132个）

衍生指标 = 原子指标 + 维度切片。所有衍生指标以 dm_opt_qy_user_execution_record_all_d 为核心数据源。

|--------|---------|---------|---------|
| DIM001 | 新客/老客 | 1=新客, 0=老客 | is_new |
| DIM002 | 大师团 | 1=是, 0=否 | is_master |
| DIM003 | 收入分类 | 大单品/常规品 | revenue_category |
| DIM004 | 获客渠道 | 私域/公域/老带新 | channel_type |

| DIM007 | 支付渠道 | 线上/线下 | channel |

### 常用衍生指标

**按新客/老客**: D001 新客核销GMV, D002 老客核销GMV, D010 新客核销收入, D011 老客核销收入, D019 新客核销服务点数, D020 老客核销服务点数

## 三、派生指标计算规则

### 比例类指标
- 新客占比 = 新客指标值 / 总指标值
- 大单品占比 = 大单品收入 / 总收入
- 大师团占比 = 大师团收入 / 总收入
- 渠道占比 = 该渠道收入 / 总收入

### 效率类指标
- 单服务点收入（整体）= 核销收入 / 核销服务点数
- 单服务点收入（非0元）= 剔除0元后的核销收入 / 剔除0元后的核销服务点数
- 单服务点收入（非0非大师）= 剔除0元和大师团后的核销收入 / 剔除0元和大师团后的核销服务点数
- 客单价 = 核销收入 / 核销人次
- 人均核销服务点 = 核销服务点数 / 核销人数

### 时段类指标
- 时间进度达成率 = 累计实际 / (月目标 x 时间进度%)
- 环比 = (本期 - 上期) / 上期
- 较前5周均值 = (本期 - 前5周均值) / 前5周均值（周报推荐基线）

### 获客类指标
- 获客成本占收比 = 获客总成本 / 新客收入（参考 30-34%）
- 公域ROI = 新客核销收入 / 获客成本（目标 >= 1.4）
- 老带新付费转化率 = ~48%
- 老带新裂变系数 = ~2.1

## 五、重要口径说明

### 关于0元单
- 0元单占比实际值18-24%，行业水平5-15%
- 0元单上升 -> 稀释单服务点收入
- 0元单 + 高值耗材 -> 毛利被直接拖累

### 关于客单价三层口径
- **整体**: 含0元、含大师团
- **非0元**: 剔除0元单
- **非0非大师**: 剔除0元单和大师团（最真实质量）

### 关于价格结构
- 定价/活动价/大促价理想比例: 5:3:2
- 实际常见: 2:3:5（需逐步优化）
- 院内改价影响约9%

### 关于核销周期
- 双11/618低价订单可能持续核销2-3个月
- 期间拉低客单价，需标注受大促订单核销影响

### 关于老客分层
- 活跃老客: 近期有到院和核销（收入贡献85%+）
- 非活跃老客: 较长时间未到院（~10%）
- 当月核销新客: 本月新建档且当月核销（~5%）

### 关于大区收入占比参考
- 华东: ~34% | 华北: ~31% | 华南: ~21% | 华中: ~14%

## 六、常用公式汇总

| 公式 | 场景 |
|------|------|
| 收入 = 核销人次 x 客单价 | 量价拆解（人视角）|
| 收入 = 核销服务点 x 单服务点收入 | 货盘拆解（货视角）|
| 医美收入 = 新客收入 + 老客收入 | 人群拆解 |
| 时间进度达成率 = 实际/（目标x时间进度%）| 进度判断 |
| 毛利率 = (收入 - 供应链成本)/收入 | 利润质量（目标48%）|
| 活跃到院率 = 当月到院活跃用户/活跃用户池 | 老客运营（目标16-17%）|
| 获客成本占收比 = 获客总成本/新客收入 | 获客效率（参考30-34%）|
| 公域ROI = 新客核销收入/获客成本 | 投放效率（目标>=1.4）|
| 升单率 = 升单人数/总到院人数 | 门店转化 |
| 品项渗透率 = 品项核销人数/总核销人数 | 品项覆盖 |
