### 一、毛利率计算口径与核心公式

品项毛利率的计算在2026年经历了三次口径变更，最终稳定为以下公式体系：

**核心公式：**
品项毛利率 = SUM(gross_margin_amt) / SUM(NULLIF(exe_income, 0))，必须用聚合计算，不可行级计算（1月15日钟明确认的BUG修复）。品项毛利额 = 核销收入(exe_income) - 耗材成本(ware_cost) - 医护手工费(manual_fee)。其中耗材成本仅包含高值耗材（consumable_type IN (1,3)），这是与财务口径的核心差异点——财务计算了全部耗材成本（含低值耗材），导致数仓口径毛利率通常比财务口径高1-4个百分点。

**让利/补贴相关公式：**
价格让利率 = max(券前GMV - exe_amount - cash_back, 0) / (base_price * exe_cnt)；用户运营额度率 = (split_new_coupon_deposit + split_coupon_deposit) / exe_income；门店改价额度率 = split_store_discount / exe_income。

**品类归类规则（互斥优先级）：**
大师团优先匹配（item_name LIKE '大师团-%' 或 revenue_category = '大师团'），其次绿标品（品项名在32-35个绿标品清单内），其余归为常规品。

**口径变更时间线：**
1月6日前为原始口径（含全部耗材），1月6-9日变更为"只包含高值耗材成本"，2月26日确认订单划扣和成本的日期计算保持原样，5月8日正式定位数仓与财务差异根因（高值 vs 全量耗材）。

### 二、数仓表与SQL参考

#### 核心数据表

`soyoung_dw.dws_opt_qy_gross_margin_stats_all_d` 是品项毛利率统计主表，由钟明于1月14日上线，粒度为天 x 品项 x 门店，关键字段包括 exe_income（核销收入）、gross_margin_amt（毛利额）、ware_cost（耗材成本）、manual_fee（医护手工费）、exe_cnt（服务点数）。

`soyoung_dw.dwd_ord_order_qy_execution_record_cost_all_d` 是耗材成本底表，1月26日上线，记录执行单级别的耗材消耗明细，含 third_product_code（耗材编码）、exclude_tax_amount（不含税成本）、consumable_type（耗材价值类型）、status（1=正常，2=撤销）。

`soyoung_dw.dm_opt_qy_user_execution_record_all_d` 是用户核销明细表，承载让利/补贴分析数据，含 base_price、exe_amount、cash_back、split_new_coupon_deposit、split_coupon_deposit、split_store_discount 等字段。

其他辅助表包括：dwd_opt_qy_per_income_detail_all_d（最细颗粒度业绩流水）、dwd_opt_qy_store_crm_per_stats_all_d（含费用分摊的业绩费用明细）、dim_sphere_bitable_human_cost（人力成本维表）、dm_opt_qy_old_bring_new_cost_all_d（老带新成本表）、haowu_jiaoyi_v1（有赞好物交易数据）、dm_private_haowu_pay_gmv（私域好物支付GMV）。

#### 典型SQL示例

品项毛利率查询：
```sql
SELECT SUM(gross_margin_amt) / SUM(NULLIF(exe_income,0)) AS 毛利率
FROM soyoung_dw.dws_opt_qy_gross_margin_stats_all_d
WHERE dp = DATE_SUB(CURRENT_DATE(),1)
  AND stat_date = DATE_SUB(CURRENT_DATE(),1)
```

低值耗材明细查询（5月8日西岚提供）：
```sql
SELECT *, not_expand_exclude_tax_amount AS 成本
FROM soyoung_dw.dwd_ord_order_qy_execution_record_cost_all_d a
WHERE a.dp = '2026-04-30'
  AND SUBSTR(create_time,1,7) = '2026-04'
  AND a.status = 1
  AND consumable_type NOT IN (1,3)
```

### 四、重大事件时间线

**1月 — 奠基月：** 1月7日与yt/gf召开毛利沟通会，确立P0级需求。1月14日钟明上线毛利统计表。1月15日蕉太狼发现毛利率=100%的BUG（收入与毛利额相同），钟明修正为聚合计算。1月21日大师团毛利率剧烈波动（69%→40%），定位为品项标签维护错误。1月30日楠木上线门店盯盘看板，新增毛利率指标。

**2月 — 扩展月：** 2月4日正式命名指标为"品项毛利率"。2月7日品项毛利率纳入26年KPI考核体系。2月26日接到新CFO需求：要求从收入中剔除"赠单冲销"。

**3月 — 深运营月：** 3月6日成都晶融汇店毛利率异常达158%，3月16日成都大悦城店出现负毛利（-2.62%）。3月26日提出Q2产研需求：耗材划扣成本链路完善、编码标准化。3月31日财务质疑3月毛利率偏高（54.16%），要求提供明细。

**4月 — 质检月：** 4月7日管理层发现日报出现负毛利品项，豆包提出"重塑经营分析底座"。4月15日Q2 OKR确定"业财一体数据基建"方向，目标差异控制在0.5%。4月17日毛利率数据突变（51%→63%），怀疑表刷新不完全。4月18日好物日报严重BUG，紧急暂停机器人。

**5月 — 成本治理月：** 5月7日财务结账毛利率49.9%，与数仓差异较大。5月8日钟明定位核心差异：数仓只算高值耗材，财务算全部耗材。5月18日豆包报告"今天毛利全是错的"。5月27日西岚编写完整SQL汇总毛利数据。

**6月 — 系统集成月：** 6月2日西岚发现划扣记录表 order_id 和 product_id 全部为0（研发BUG），正在紧急沟通。同日楠木提出院务系统门店目标下发功能存在重复改造风险。

### 五、数据质量问题与排查记录

**已解决：**
毛利率=100%的BUG（1月15日，SUM聚合修复）；大师团标签错误致毛利率波动（1月21日，需禾叶整体维护基础品标签）；撤销状态订单无成本（1月26日，底表补充status=2）；门店入库成本字段不准（1月26日，调整为数仓汇总加工逻辑）。

**持续跟踪：**
数仓与财务口径差异（5月8日定位，高值耗材 vs 全部耗材，尚未统一）；晶融汇等门店毛利率反复异常（3月6日/22日，需逐单排查）；医护手工费无法拆分医生/护士（3月10日，表结构限制）；耗材进销存单位/编码/名称不统一（3月26日，列入Q2产研需求）；划扣记录表关键字段为0（6月2日，研发沟通中）。

**三大关键断裂点（飞书文档梳理）：**
品项映射断裂——品项服务与耗材无唯一关联（无"标准SOP"）；计量单位断裂——大包装拆小包装无法计算，成本差数倍；源头成本断裂——手工录入成本全是错误的。这些问题预计随用友ERP上线（6月30日）逐步解决。

### 六、关键发现与业务洞察

**补贴/让利分析：** 让利补贴已吃掉26.2%收入（1.31亿）。常规品价格让利率从3月0.5%跳升至4月5.6%（10倍），5月用户运营率达26.1%已破红线。L0/L1两个会员等级出现严重补贴倒挂（L0补贴率1884%、纯薅率68.9%），灌券策略应向高等级（L5+）倾斜。

**4月毛利率回落归因：** 从3月54.1%降至4月51.4%（-2.67pp），不是结构问题（占比效应仅-0.01pp），而是各品类自身毛利率回落——绿标品从73.6%降至68.3%（-5.33pp）是最大拖累。

**4月踩刹车清单（8个品项）：** 美国品牌（毛利率21.4%，价格让利率16.2%）、弗缦注射（毛利率42.8%，让利率22.8%）、韩国品牌（毛利率仅7.5%）、嗨体1.5+2.5（毛利率8.4%）等需要立即干预。

**奇迹胶原新品上市：** 首月毛利率仅19.7%-22.5%，远低于绿标品矩阵均值，拉低整体绿标品毛利率。
